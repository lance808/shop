# 三人分工方案

---

## A — 前台核心（用户 + 商品 + 搜索 + 前台公共）

**负责模块：用户认证、商品浏览、搜索、前台公共组件**

| 层级 | 文件 | 工作量 |
|------|------|--------|
| Controller | `UserController.java`、`ProductController.java` | 2 个 |
| DAO | `UserDao.java`、`ProductDao.java` | 2 个 |
| Mapper | `UserMapper.xml`、`ProductMapper.xml` | 2 个 |
| Service | `UserService.java` + `impl/UserServiceImpl.java`、`ProductService.java` + `impl/ProductServiceImpl.java` | 4 个 |
| Entity | `User.java`、`Product.java` | 2 个 |
| View | `header.jsp`、`footer.jsp`、`index.jsp`、`login.jsp`、`register.jsp`、`profile.jsp`、`product_list.jsp`、`product_detail.jsp`、`pagination.jsp` | 9 个 |
| CSS | `style.css`（前台样式） | 1 个 |
| Interceptor | `LoginInterceptor.java` | 1 个 |

**总计：23 个文件**

> 技术点：`@RequestMapping` 路由、Model 传参、`@RequestParam` 参数绑定、分页查询、JSTL 标签、CSS Grid 布局

---

## B — 交易流程（购物车 + 订单 + 收货地址 + 工具类）

**负责模块：购物车 CRUD、下单流程、收货地址管理、密码/订单号工具**

| 层级 | 文件 | 工作量 |
|------|------|--------|
| Controller | `CartController.java`、`OrderController.java`、`AddressController.java` | 3 个 |
| DAO | `CartDao.java`、`OrderDao.java`、`OrderItemDao.java`、`AddressDao.java` | 4 个 |
| Mapper | `CartMapper.xml`、`OrderMapper.xml`、`OrderItemMapper.xml`、`AddressMapper.xml` | 4 个 |
| Service | `CartService.java` + `impl/CartServiceImpl.java`、`OrderService.java` + `impl/OrderServiceImpl.java`、`AddressService.java` + `impl/AddressServiceImpl.java` | 6 个 |
| Entity | `Cart.java`、`Order.java`、`OrderItem.java`、`Address.java` | 4 个 |
| View | `cart_list.jsp`、`order_confirm.jsp`、`order_list.jsp`、`order_detail.jsp`、`address_list.jsp`、`address_form.jsp` | 6 个 |
| Util | `PasswordUtil.java`、`OrderNoUtil.java` | 2 个 |

**总计：29 个文件**

> 技术点：`@Transactional` 事务、库存扣减、金额计算、多表关联查询、MD5 加密、订单号生成

---

## C — 后台管理 + 扩展功能（管理端 + 收藏 + 评价 + 统计）

**负责模块：后台商品/分类/订单管理、数据统计仪表盘、收藏夹、商品评价**

| 层级 | 文件 | 工作量 |
|------|------|--------|
| Controller | `AdminController.java`、`FavoriteController.java`、`ReviewController.java` | 3 个 |
| DAO | `CategoryDao.java`、`FavoriteDao.java`、`ReviewDao.java` | 3 个 |
| Mapper | `CategoryMapper.xml`、`FavoriteMapper.xml`、`ReviewMapper.xml` | 3 个 |
| Service | `CategoryService.java` + `impl/CategoryServiceImpl.java`、`FavoriteService.java` + `impl/FavoriteServiceImpl.java`、`ReviewService.java` + `impl/ReviewServiceImpl.java` | 6 个 |
| Entity | `Category.java`、`Favorite.java`、`Review.java` | 3 个 |
| View（后台） | `admin/header.jsp`、`admin/footer.jsp`、`admin/index.jsp`、`admin/stats.jsp`、`admin/product_list.jsp`、`admin/product_form.jsp`、`admin/category_list.jsp`、`admin/category_form.jsp`、`admin/order_list.jsp`、`admin/order_detail.jsp` | 10 个 |
| View（扩展） | `favorite_list.jsp`、`review_list.jsp` | 2 个 |
| CSS | `admin.css` | 1 个 |
| Interceptor | `AdminInterceptor.java` | 1 个 |

**总计：32 个文件**

> 技术点：图片上传 `MultipartFile`、聚合统计 SQL（`SUM`/`COUNT`/`AVG`/`GROUP BY`）、父子分类级联、`@RequestMapping` method 限定

## 工作量对比

| 成员 | 文件数 | 核心职责 |
|------|--------|---------|
| **A** | 23 | 前台核心：用户 + 商品展示 + 搜索 + 前台 UI |
| **B** | 29 | 交易链路：购物车 + 下单 + 收货地址 + 工具 |
| **C** | 32 | 后台管理：管理端 + 收藏 + 评价 + 数据统计 |

> 三人工作量均衡，B 交易逻辑最重（事务 + 多表），C 文件最多但单文件较轻（CRUD 为主），A 居中且涉及分页等通用组件。
